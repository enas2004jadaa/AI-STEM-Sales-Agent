# AI-STEM-Sales-Agent
AI-powered Telegram sales assistant built with n8n, OpenAI, and Google Sheets for answering customer questions, retrieving knowledge from a structured knowledge base, and capturing qualified leads.

Overview

This project is an AI-powered Telegram sales assistant built using n8n, OpenAI GPT-4o Mini, and Google Sheets.

The workflow answers customer questions about STEM programs, retrieves information from structured knowledge bases, and automatically records qualified leads.

Workflow

User sends a Telegram message.
AI Agent receives the request.
Searches the appropriate Knowledge Base.
Generates an accurate response.
If the user wants to register:
Collects required information.
Saves the lead to Google Sheets.
Sends the response back to Telegram.
